# Pixel Portfolio

A personal portfolio and blog built with Next.js, React and Tailwind CSS, featuring a pixel-art visual style to showcase projects and posts.

## Features

- Hero section highlighting you or your main project
- About section with background, skills and experience
- Services section to list offerings or specialties
- Works gallery with individual project pages
- Blog with article list and detail pages
- Responsive design and light/dark theme support

## Tech Stack

- Next.js 13 (App Router)
- React 18
- TypeScript
- Tailwind CSS
- Radix UI, Framer Motion, Lucide icons

## Requirements

- Node.js 18+ (recommended)
- npm or yarn

## Quick Start

1. Install dependencies

```bash
npm install
# or
yarn install
```

2. Run the dev server

```bash
npm run dev
```

3. Build for production

```bash
npm run build
npm start
```

The app runs on http://localhost:3000 by default.

## Available Scripts

- `dev` — start development server (`next dev`)
- `build` — build production artifacts (`next build`)
- `start` — start production server (`next start`)
- `lint` — run linter (`next lint`)

## Deployment

Deploy to Vercel for the best Next.js experience, or any Node.js hosting provider.
